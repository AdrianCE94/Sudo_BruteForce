# Sudo_BruteForce
Script hecho en bash para realizar un ataque de fuerza bruta a un usuario de un sistema Linux.

Se ejecuta de la siguiente forma:

```
bash Linux-Su-Force.sh mario /usr/share/wordlists/rockyou.txt
```
